# AI Vocabulary Generator for Anki

This add-on generates vocabulary flashcards using AI.

## Features
- Multi-word input
- JSON-based output
- Custom Vocabulary Card
- Deck selection
- HyperTTS compatible

## Setup
1. Install the add-on
2. Open Tools → Add-ons → Config
3. Set your OpenAI API key
4. Restart Anki

## Usage
Tools → Generate Flashcards (Multi-Word)

## Requirements
- Anki 25+
- OpenAI-compatible API
